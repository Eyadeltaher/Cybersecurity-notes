A comprehensive collection of **web application security notes, testing methodologies, payloads, scripts, tools, write-ups, and curated wordlists**.

The repository is built while studying web application penetration testing, solving PortSwigger Web Security Academy labs, participating in CTFs, and researching real-world vulnerabilities.

| **Authentication** | Authentication vulnerabilities, session management |

| **Race Condition** | Race condition vulnerabilities and testing |

| **Cross-Site Scripting (XSS)** | Stored, Reflected, DOM XSS, payloads |

| **Cross-Origin Resource Sharing (CORS)** | Misconfigurations and exploitation |
| **XML External Entity (XXE)** | XXE attacks and payloads |
| **Server-Side Template Injection (SSTI)** | SSTI exploitation |
| **Server-Side Request Forgery (SSRF)** | SSRF methodology and payloads |

| **File Upload** | File upload vulnerabilities & bypass techniques |

| **Insecure Deserialization** | Deserialization vulnerabilities |

| **LFI / RFI** | Local & Remote File Inclusion |

| **Open URL Redirect** | Open Redirect vulnerabilities |
| **Clickjacking** | UI redressing attacks |
